# AlgoViz+ Android Development Baseline

## Required Tools

- **JDK**: 17.0.10 (Azul Zulu / Oracle)
- **Android Studio**: Hedgehog | 2023.1.1 or later
- **Gradle**: 8.6
- **Android Gradle Plugin**: 8.3.0
- **Kotlin**: 1.9.22
- **Min SDK**: 24
- **Target SDK**: 34
- **Compile SDK**: 34

## Installation Commands

```powershell
# Verify JDK
java -version

# Verify Android SDK
$env:ANDROID_HOME

# Install Gradle (if not using wrapper)
choco install gradle --version=8.6
```

## Android SDK Components Required

- Android SDK Platform 34
- Android SDK Build-Tools 34.0.0
- Android Emulator
- Google Play Services
